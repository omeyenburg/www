Scrolling

A quality-of-life plugin for the Markdown editor Obsidian providing many scrolling related enhancements you cannot live without:

  • Keep the text cursor centered
  • Disable code block wrapping & allow scrolling
  • Prevent MathJax overflow & allow scrolling
  • Prevent file name cut-off & allow scrolling
  • Remember & restore the position inside files on open
  • Zoom images by scrolling
  • Customize maximum line length
  • Use Vim keybinds to scroll in reading mode
  • Customize mouse & touchpad scroll speed
